[机器学习] 线性回归问题一

Posted on 2019-02-05

Regression

Step1 Model (定义一个模型)

我们先要根据需定义一个方法集合（a set of function）。

例如: $f(x) = y = x \times w+b$ （一般我们称y为输出，x为特征或输入，w为权重,b为bias）

在解决实际问题中特征往往不止一个，所以方程式也可以写作

$f(x) = y = b + \sum w_i x_i$

简单来说我们需要做的就是在有限的样本中,通过穷举例不同的w和b来找到一个 $f^*$ (期望函数) 能够尽可能的拟合已知样本和预测未知样本。

Step2 Goodness of Function (判断方法的好坏)

要想衡量我们找到的期望函数是否足够好我们就要先找到一个Loss Function L ,它的输入是我们找的期望函数，它的输出是这个函数到底有多 “不好“

$L(f) = L(w ,b)
= \sum_{n=1}^{k} (y^n - (w*x^n+b))^2
$

Step3 Best Function (选出最好的方法)

我们需要找到一个$f^*$让$L(w,b)$的值最小，表达式可以写作:

$f^* = arg$ $min$ $L(f)$

$w^ , b^ = arg$
$min$ $L(w,b)$

=$arg$ $min$ $\sum^{k}_{n=1}(y^n-(w*x^n+b))^2​$

那么如何找到这个$f^*$呢？在下一个文章中再介绍Gradient Descent(梯度下降算法)
